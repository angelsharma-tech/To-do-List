# 📝 To-Do List Application (Python + Tkinter)

## 📌 Overview

This is a simple and user-friendly **To-Do List desktop application** built using Python and the Tkinter library. It helps users manage daily tasks efficiently through an interactive graphical interface.

The application supports adding, updating, deleting, and marking tasks as complete, along with saving tasks for future use.

---

## ✨ Features

* ➕ Add new tasks
* 📝 Update existing tasks
* ❌ Delete selected tasks
* 🗑️ Delete all tasks with confirmation
* ✅ Mark tasks as complete (color indication)
* 💾 Save tasks to a file (`tasks.txt`)
* 🔄 Load saved tasks automatically on startup
* ⚠️ User-friendly alerts and confirmations

---

## 🖥️ Tech Stack

* **Library:** Tkinter (for GUI)

  * Event handling

## 🚀 How to Run

1. Make sure Python is installed on your system.
2. Clone this repository:

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
3. Navigate to the project folder:

   ```bash
   cd your-repo-name
   ```
4. Run the application:

   ```bash
   python ToDoList.py
   ```
